# Highest Product

Given an array of `N` integers:
find the highest product by multiplying `3` elements.

Constraints
===========
* <code>3 <= N <= 5*10<sup>5</sup></code>

Examples
--------

### Example 1
Input: [1, 2, 3, 4]
Output: 24
Explanation:
Output = 2 * 3 * 4

### Example 2
Input: [0, -1, 10, 7, 5]
Output: 350
Explanation:
Output = 5 * 7 * 10
